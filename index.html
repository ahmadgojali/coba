About Me

Halo, perkenalkan nama saya Ahmad Gojali, umur 23 tahun, saya tinggal di daerah Tambelang, Bekasi, Jawa barat. Background pendidikan saya, saya berkuliah di Universitas Gunadarma mengambil jurusan sistem informasi. Dimana pada jurusan tersebut lebih berkonsentrasi pada pengembangan perangkat lunak bisnis dan cara menganalisis kebutuhan perangkat lunak dan teknologi informasi pada dunia bisnis. Pada bulan Oktober 2018 yang lalu saya telah menyelesaikan pendidikan saya di perguruan tinggi tersebut dan mendapatkan gelar sarjana komputer yang ditempuh dalam waktu 4 tahun dengan IPK 3.45.
Sebelumnya saya juga sekolah menengah kejuruan di salah satu sekolah negeri di tambelang, jurusan yang saya ambil adalah Teknik Komputer dan Jaringan. Di sinilah awalnya saya tertarik dan menyukai dunia teknologi dan komputer, saya tau bagaimana caranya troubleshooting komputer, membangun sebuah jaringan komputer, membangun sebuah server dan membuat website sederhana. Dan ilmu yang saya dapat di SMK tersebut menjadi modal saya untuk melanjutkan studi saya di Universitas Gunadarma.
Pengalaman saya bekerja di dunia IT dimulai ketika saya diterima di Laboratorium manajemen menengah Universitas Gunadarma, yang pada saat itu sedang membutuhkan seorang programmer untuk mengerjakan tugas-tugas yang berhubungan dengan IT. Saya memulai bekerja di laboratorium sejak agustus 2016 sampai dengan september 2018.

Dengan kesempatan tersebut saya bisa mengembangkan hardskill saya terutama di web programming, selain hardskill saya juga belajar tentang softskill, tentang bagaimana cara berorganisasi, bekerja secara tim maupun individu, mampu bekerja dengan deadline dan mampu mengkoordinasikan junior programmer.
Jobdesk yang saya kerjakan ketika menjadi seorang asisten programmer adalah :

  • Maintenance Komputer
  • Troubleshooting hardrware dan software
  • Sebagai Admin Web
  • Mengembangkan Website Virtual-lab dengan CodeIgniter
  • Merancang dan membuat aplikasi penerimaan asisten baru menggunakan PHP dan MySQL
  • Mengelola data dan nilai praktikan menggunakan database MySQL
  • Membuat modul animasi pembelajaran

Karena passion saya di dunia IT, saya berharap mendapatkan pekerjaan yang sesuai di bidang saya, dengan ilmu dan pengalaman yang saya miliki sebelumnya serta latar belakang pendidikan saya, saya yakin dapat berkontribusi untuk kemajuan perusahaan kedepannya.
